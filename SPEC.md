# 한국노마드 웹사이트 개선 실행 계획

## 프로젝트 개요

현재 Next.js 14 기반 디지털 노마드 도시 찾기 플랫폼을 단순화하고, 핵심 기능에 집중하도록 개선합니다. 데이터베이스 없이 Mock 데이터를 사용하여 점진적으로 기능을 개선합니다.
